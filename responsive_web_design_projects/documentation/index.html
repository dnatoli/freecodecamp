Variables

        x = 1
        name = "Dante"
        x = name

Variables in Python are not statically typed. You don't need to declare the type of object you are referring to with a variable.

Python variables are dynamically typed, so you can change the type of object being pointed to by a single variable.

Python variables can be

  • integers
  • floating point numbers
  • strings
  • objects
  • functions

Functions

        def foo(bar):
            print(bar)

Functions in Python do not specify a return type or a type for the arguments. Scope is maintained by consistent indentation of four spaces.

Control Flow

        
        if not foo:
            print("False")
        elif foo:
            print("True")
        else:
            print("This can't happen")
        
        
        for foo in foos:
            print(foo)
        for i in range(len(foos)):
            print(foos[i])

Control flow statements are scoped similarly to functions. Python "for" loops use iterators by default, so to use indexes, you instead make a range array.

Lists and Tuples

                
        list = [1, 2, 3]
        list += [4, 5]
        list.append(6)

        [print(i) for i in list]

        tuple = (1, 2)
        tuple2 = tuple + (3, 4)

Lists and tuples are the main array structure in Python. Tuples are immutable, so once they are created you can't change them and must create a new one if needed. There are not restrictions on the types of objects put into a list or tuple, and they can contain more than one type of object at the same time.

Lists are mutable and can have elements added or removed with a variety of methods. List comprehensions are a way of iterating over collections using a concise syntax.

Classes

                
        class Foo:
            def __init__(self):
                self.bar = "bar"
            def doThing(self):
                print(self.bar)
        
        foo = Foo()
        foo.doThing()

Python is an object-oriented language, so it has a syntax for defining classes to encapsulate data and behavior.

Instance methods require "self" as the first parameter, and are called with standard "." syntax. Fields/instance variables are declared in the initializer method and are public.

Lambdas

                
        foo = lambda x: print(x)
        foo("Hello world")

Python lambda function syntax is pretty simple.

You can include multiple parameters, capture variables in scope, and assign to a variable.
